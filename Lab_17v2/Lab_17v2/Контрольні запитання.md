# **Контрольні запитання**

## **1. Як і навіщо використовуються структури?**

Структури застосовуються для об’єднання кількох логічно пов’язаних даних в один тип.
Вони дозволяють створювати складні об’єкти (наприклад, студент, книга, товар), роблять програму зручнішою та покращують читабельність.
Структури використовуються коли потрібні:

* компактні об’єкти без складної логіки;
* швидке створення та копіювання значень;
* масиви або колекції складних типів даних.

---

## **2. Як описати структуру? Де це можна зробити?**

Структура описується за допомогою ключового слова `struct`.

Приклад:

```csharp
struct Student
{
    public string Surname;
    public string Name;
    public string Patronymic;
}
```

Опис структури можна розмістити:

* **у просторі імен (namespace)**, але поза класом `Program` — це найкраще місце;
* або всередині класу як вкладений тип.

---

## **3. Чим структури відрізняються від масивів?**

* **Структура** — це користувацький тип даних, який містить кілька полів різних типів.
  Вона описує **один об’єкт** (один студент, одна книга тощо).

* **Масив** — це набір елементів **одного типу**.
  Наприклад: `int[]`, `string[]`, `Student[]`.

Тобто структура — це опис об’єкта, а масив — множина таких об’єктів.

---

## **4. Що таке ініціалізатор? Який синтаксис його використання для структури?**

Ініціалізатор — це спосіб одразу встановити значення полів структури під час її створення.

Приклад:

```csharp
Student st = new Student
{
    Surname = "Іванов",
    Name = "Іван",
    Patronymic = "Іванович"
};
```

Також ініціалізатор можна використовувати у масивах:

```csharp
Student[] group =
{
    new Student { Surname="Андрусь", Name="Петро", Patronymic="Олегович" },
    new Student { Surname="Коваль", Name="Марія", Patronymic="Ігорівна" }
};
```

---

## **5. Як ініціалізувати структуру значеннями за замовчуванням?

Як називаються методи для ініціалізації полів структури? Скільки їх може бути?**

Щоб отримати значення за замовчуванням, достатньо створити структуру:

```csharp
Student st = new Student();
```

Усі поля отримають стандартні значення:

* `0` — для числових типів,
* `false` — для `bool`,
* `null` — для `string`.

Методи, які ініціалізують поля структури, називаються **конструкторами**.

Особливості:

* У структурі **дозволено створювати будь-яку кількість параметризованих конструкторів**.
* **Заборонено оголошувати конструктор без параметрів** — він створюється автоматично компілятором.
* Конструктор може встановлювати значення полів вручну або зчитувати їх з консолі.
